Menu Restrito

O Menu Restrito é um sistema web desenvolvido para auxiliar pessoas com restrições alimentares, especialmente aquelas com intolerância ao glúten e à lactose.
A plataforma permite que usuários encontrem alimentos e estabelecimentos seguros, com informações claras sobre ingredientes, modos de preparo e possíveis contaminações cruzadas.
Além disso, os estabelecimentos podem cadastrar seus cardápios e informar filtros de restrição, tornando o ambiente mais inclusivo e transparente.



Funcionalidades Principais

Cadastro e login de usuários (cliente e estabelecimento)

Busca de produtos e estabelecimentos com filtros por tipo de restrição

Exibição de informações detalhadas sobre ingredientes e preparo

Sistema de avaliação e feedback dos usuários

Gerenciamento de perfis e dados dos estabelecimentos


Tecnologias Utilizadas

HTML5 e CSS3 — Estrutura e design responsivo

JavaScript (ES6+) — Lógica e interatividade do sistema

Node.js e Express.js — Criação do servidor e rotas da aplicação

Fetch API — Comunicação entre front-end e back-end

JSON / LocalStorage — Armazenamento local de informações


Como Executar o Projeto
1️.Clonar o repositório

Abra o Git Bash e execute:

git clone https://github.com/seu-usuario/menu-restrito.git

2️.Acessar a pasta do projeto
cd menu-restrito

3.Instalar as dependências
npm install

4.Iniciar o servidor
node server.js

5.Acessar no navegador

Abra o navegador e vá para:

http://localhost:3000



Tipos de Usuário
Cliente

Pode se cadastrar, fazer login e buscar produtos ou locais adequados às suas restrições.

Pode aplicar filtros, ver descrições e avaliar estabelecimentos.

Estabelecimento

Pode cadastrar o local e informar os tipos de restrição atendidos.

Pode editar suas informações e responder avaliações de clientes.



Considerações Finais

O Menu Restrito busca promover inclusão alimentar e segurança no consumo de alimentos, usando a tecnologia como aliada para facilitar a vida de pessoas com restrições.
Com uma interface simples e intuitiva, o projeto oferece uma solução prática para quem precisa encontrar opções seguras e confiáveis.
